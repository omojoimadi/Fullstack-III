# Learning Outcomes

- Understand SPA Routing: Learn how client-side routing differs from traditional multi-page applications
- BrowserRouter: Wrap application to enable routing functionality using HTML5 History API
- Routes & Route: Structure and define URL paths and their corresponding components
- Link Component: Create navigation without triggering full page reloads
- Dynamic Navigation: Implement seamless page transitions while maintaining application state
- 404 Handling: Catch undefined routes using wildcard path (*)
